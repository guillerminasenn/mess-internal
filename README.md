# MESS: Multiproposal Elliptical Slice Sampling

Python implementation of Multiproposal Elliptical Slice Sampling (MESS), a self-tuning multiproposal MCMC algorithm for Bayesian inference with Gaussian priors.

## Quick Start

See the `notebooks/` directory for examples.

## Features

- MESS algorithm including its angular and Euclidean variants. 

- Built-in problems: GP regression, logistic regression, and two Bayesian inverse problems (semi-blind deconvolution, and toy model for solute transport). 

- ESS/MSJD diagnostics for diagnosing mixing.
